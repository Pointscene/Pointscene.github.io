WorkflowTask

Type for WorkflowTask.
Workflow tasks are used to defined the workflow graph.
It is possible to build custom workflows by chaining different tasks.
See workflow tasks for more documentation and examples.

Field Name Description
name - object

Define the task name.
The task name needs to be unique in the workflow context.
The name is used as input for consecutive tasks in the workflow graph.
type - object

Define task type e.g. pdal.info.
See tasks for available task types.
args - object

Task args.
See tasks for available task args.
inputs - object

Define task inputs.
Use task names as input values e.g. ["download"].
Example
{
  "name": "xyz789",
  "type": "abc123",
  "args": {},
  "inputs": ["xyz789"]
}
